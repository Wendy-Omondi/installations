# **Install `PEP8`**

## **Using `Pip3`**

**Install pip3**

`$ sudo apt-get install python3-pip`

**Install `Pep8`**

```
$ sudo apt-get install python3-pep8
$ sudo apt-get install python3-pip
$ sudo pip3 install -Iv pep8==1.7.0
```

Make sure you have the right version

```
$ pep8 --version
1.7.0
$
```
